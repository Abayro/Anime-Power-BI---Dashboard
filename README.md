# Anime Analytics Dashboard | Power BI

## Project Overview
This Power BI dashboard provides insights into anime trends, ratings, popularity, and audience engagement. The dashboard helps users explore anime performance through interactive visualizations and KPI metrics.

## Objectives
- Analyze anime ratings and popularity
- Identify top-rated anime
- Explore anime distribution by type
- Track anime releases over time

## Dashboard Features
- Total Anime KPI
- Average Rating KPI
- Total Members KPI
- Anime by Type (Donut Chart)
- Anime Releases by Year
- Top Rated Anime

## Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset

## Key Insights
- TV series represent the largest share of anime content.
- Highly rated anime tend to attract larger member communities.
- Anime production has increased significantly over the years.
- Interactive filters allow deeper exploration of anime trends.
